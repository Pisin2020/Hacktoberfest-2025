# TypeScript Snippets

## Example HTTP Server

An example of fully typed Node.JS HTTP server.

## Setup

### Dependencies

```bash
npm install
```

### Development (HTTP Server)

```bash
npm run http
```

### Build

```bash
npm run build
```

### Test

```bash
curl http://localhost:3000/
```

## Environment

`.env` File

```
HTTP_PORT = 3000
HTTP_HOST = "localhost"
```
