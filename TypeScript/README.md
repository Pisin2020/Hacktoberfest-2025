# TypeScript Snippets

## Example HTTP Server

An example of fully typed Node.JS HTTP server.

## Setup

### Dependencies

```
npm install
```

### Development (HTTP Server)

```
npm run http
```

### Build

```
npm run build
```

## Environment

`.env` File

```
HTTP_PORT = 3000
HTTP_HOST = "localhost"
```
